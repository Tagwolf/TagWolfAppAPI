# TagWolf - Aus Kronach

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![Python](https://img.shields.io/badge/python-3.8+-green)
![License](https://img.shields.io/badge/license-MIT-orange)

Eine moderne Desktop-Anwendung mit lila-blauem Farbverlauf, Animationen und API-Integration.

## Features

- 🎨 **Lila-blauer Farbverlauf** - Modernes Design
- ✨ **Animationen** - Schwebende Partikel, Pulsierender Text, Glow-Effekte
- 🔌 **API-Integration** - Verbindung zu externen Diensten
- 💻 **Eigenständige EXE** - Keine Python-Installation nötig
- 🎯 **Zentrierte Ansicht** - Perfekte Platzierung auf jedem Bildschirm
- 🚀 **Schnell und leicht** - Minimale Systemanforderungen

## Installation

### Methode 1: Fertige EXE

1. `TagWolf.exe` herunterladen
2. `install_tagwolf.bat` ausführen
3. Desktop-Verknüpfung verwenden

### Methode 2: Von Source

```bash
git clone https://github.com/tagwolf/TagWolf.git
cd TagWolf
pip install -r requirements.txt
python src/main.py
